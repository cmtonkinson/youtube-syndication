# Personas

## Primary Persona
- Who they are: Home media server operator curating YouTube into Plex.
- What problem hurts the most: Manual download/rename/metadata work and
  inconsistent Plex ingestion.
- What success looks like: One command keeps the library updated with correct
  ordering, naming, and metadata.
- What they do *not* care about: A rich UI, cloud hosting, or multi-user
  workflows.

## Secondary Personas
- Power user maintaining a shared Plex library who values consistency and
  storage predictability.
- Tinkerer who schedules syncs and wants controllable skip filters.

## Explicit Non-Goals / Anti-Personas
- We are NOT building this for: Content creators or public distribution.
- We will NOT optimize for: Real-time streaming or on-demand downloads.
